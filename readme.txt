Add reviews and testimonials to your website with ease

== Description ==

The InnovaReviews plugin for Wordpress allows you to easily add reviews to your Wordpress site.

== Installation ==

1. Install and activate the plugin.
2. Go to https://www.innovareviews.com/signup and create an account.
3. Copy or write down your client code from the InnovaReviews dashboard.
4. Go back to your Wordpress InnovaReviews settings page and paste or enter your client code.

== Frequently Asked Questions ==

= Do I need to create an account with InnovaReviews =
Yes, you will need to sign up at https://www.innovareviews.com/signup to use this plugin.
